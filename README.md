# Zerodha Clone (Frontend)

This is a **frontend clone** of the [Zerodha](https://zerodha.com/) website built using **React** and **Bootstrap**.

## Features
- Replicates the main landing page layout of Zerodha.
- Responsive design for desktop and mobile screens.
- Navigation bar with links to main sections.
- Interactive components like cards, banners, and footer sections.
- Built entirely with **React functional components** and **Bootstrap styling**.

## Tech Stack
- **React.js** – for building UI components
- **Bootstrap** – for styling and responsive layout
- **CSS** – custom styles for additional design

   git clone https://github.com/taniasha/Zerodhaa.git
2. Navigate to project folder:
